# cryptx-dashboard-

The cryptx dashboard is a single page webpage that displays an exact redesign of a figma designed dashboard. With a header, sidebar and a static btc chart.

Technologies used:

Tailwind CSS: Used for the styling of each individual component
Recharts was used to help design the static Bitcoin chart
React was the program used to build the whole project.
React hooks were also used.
Lucide react was used for icons on the sidebar and metriccards.
Vercel for live deployment.

Features used:
Static Bitcoin chart.
Fixed sidebar and header.
Reusable components for displaying the crypto statics on the metriccards.
Tailwind CSS was used to style the whole webpage.
